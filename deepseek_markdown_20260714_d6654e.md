# 💪 BMI & Fitness Toolkit

A comprehensive health and fitness tracking web application built with Streamlit.

## ✨ Features

- 📊 **BMI Calculator** - Metric/imperial support, healthy weight ranges, diet suggestions
- 🔥 **Fitness Calculator** - BMR, TDEE, calorie goals, macros, water intake, body fat
- 💪 **Workout Planner** - Personalized plans based on BMI and goals
- 📜 **History Tracking** - Save and view all BMI calculations
- 📈 **Progress Trends** - Visualize BMI changes over time

## 🛠️ Tech Stack

- [Streamlit](https://streamlit.io/) - Web framework
- [Pandas](https://pandas.pydata.org/) - Data processing
- [NumPy](https://numpy.org/) - Numerical computations
